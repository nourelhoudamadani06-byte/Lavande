# Lavande
La lavande est un arbrisseau aromatique vivace, connu pour ses fleurs bleues ou violettes en épis et son parfum agréable. Elle pousse naturellement dans des milieux secs, en particulier dans le bassin méditerranéen, et est largement cultivée pour ses huiles essentielles.
